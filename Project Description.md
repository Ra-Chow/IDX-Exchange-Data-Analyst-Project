# IDX-Exchange-Data-Analyst-Project

#### The IDX Exchange Data Analyst Internship is a structured 12-week program designed to take raw, confidential housing market transaction datasets and build a data pipeline.

#### Phase 1: Data Cleaning: Aggregating monthly files and removing invalid records.
#### Phase 2: Market Analytics: Engineering key housing metrics (e.g., price per square foot, days on market, etc.)
#### Phase 3: Competitive Intelligence: Identifying top real estate agents and offices.
#### Phase 4: Dashboard Development: Building interactive tracking tools in Tableau.
#### Phase 5: Market Insights: Presenting my final findings to the IDX Exchange Data Analyst Summer 2026 cohort.
